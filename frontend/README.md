# Olive Press Frontend

React + TypeScript frontend for the Olive Press Management System.

## Run

```bash
npm install
npm run dev
```

Default API base URL:

```bash
http://localhost:8080
```

Override it with:

```bash
VITE_API_BASE_URL=http://localhost:8080
```

## Build

```bash
npm run build
```
